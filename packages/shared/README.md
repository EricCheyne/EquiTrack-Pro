# @equitrack/shared

Shared types, constants, and utilities for EquiTrack Pro.

## Getting Started

```bash
pnpm build
```

## Structure

- `src/types/` - TypeScript type definitions
- `src/constants/` - Application constants
- `src/utils/` - Utility functions

## Scripts

- `build` - Compile TypeScript
- `lint` - Run ESLint
